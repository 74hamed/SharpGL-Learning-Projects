# SharpGL Learning Projects

A collection of educational OpenGL projects developed using C# WinForms and SharpGL.

This repository contains multiple 3D graphics demonstrations focused on rendering, transformations, user interaction, and sensor visualization.

## Projects Included

### 1. Interactive 3D Cube Viewer

A simple OpenGL cube that can be rotated using mouse input.

Features:

* Mouse-controlled rotation
* Perspective projection
* Colorful cube rendering
* Depth testing

### 2. Interactive 3D Camera Model

A custom 3D camera model created using OpenGL primitives.

Features:

* Fully rotatable camera model
* Mouse interaction
* Multiple 3D components
* Real-time rendering

## Technologies Used

* C#
* Windows Forms
* SharpGL
* OpenGL

## Learning Topics

This repository demonstrates:

* OpenGL initialization
* Projection and model-view matrices
* 3D transformations
* Object rendering
* Mouse interaction
* Animation loops
* Sensor orientation visualization
* Depth buffering

## Requirements

* Visual Studio
* .NET Framework
* SharpGL NuGet Package

## Getting Started

Clone the repository:

```bash
git clone https://github.com/74hamed/SharpGL-Learning-Projects.git
```

Open the solution in Visual Studio and run any project.
